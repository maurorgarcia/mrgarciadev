# Mauro García | Full Stack Developer & AI Automation

Este es mi portafolio profesional, diseñado con una estética moderna "Vercel-like" y optimizado para mostrar mi trayectoria en el desarrollo de software y la automatización inteligente.

## 🚀 Sobre el Proyecto
El objetivo de este sitio es presentar mi perfil técnico a reclutadores y clientes, destacando mi formación académica en la **UTN** y mi especialización actual en el ecosistema **Full Stack**.

### Características Técnicas:
- **Estética Premium:** Uso de mesh gradients, texturas de grano y glassmorphism para una experiencia visual de alto nivel.
- **Performance:** Optimizado con Vanilla CSS y JavaScript moderno (Intersection Observer) para una carga fluida sin frameworks pesados.
- **Responsive:** Adaptabilidad total a dispositivos móviles y tablets.
- **Foco en Automatización:** Sección dedicada a proyectos que integran IA y flujos operativos eficientes.

## 🛠️ Stack Tecnológico
- **Frontend:** HTML5, CSS3 Moderno (Variables, Gradients, Flex/Grid), JavaScript ES6+.
- **Diseño:** Estética minimalista inspirada en los estándares de Vercel y Linear.
- **Educación:** Tecnicatura en Programación (UTN) & Diplomatura Full Stack (Coderhouse).

## 📂 Estructura del Repositorio
- `index.html`: Estructura semántica del sitio.
- `style.css`: Sistema de diseño y animaciones.
- `main.js`: Lógica de filtrado de proyectos y efectos de scroll.
- `images/`: Recursos visuales del portafolio.

## 🌎 Despliegue (Vercel)
Este proyecto está listo para ser desplegado en Vercel mediante sincronización directa con este repositorio de GitHub.
